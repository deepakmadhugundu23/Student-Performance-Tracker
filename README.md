# Student-Performance-Tracker
# Diagnostics Dashboard

An interactive Python dashboard built with Streamlit that tracks student academic performance and delivers automated, data-driven diagnostic feedback.

# 🚀 Features
* **Interactive Filtering:** Slice student performance data dynamically by grade, subject, and assessment type.
* **Performance Visualizations:** Rich, clean charts built with Matplotlib to track trends over time.
* **Automated Diagnostics:** A dedicated backend module that evaluates performance metrics and flags critical learning gaps automatically.

# 📁 Project Structure
* `dashboard.py` - The primary Streamlit frontend, UI layouts, and user input controls.
* `diagnostics.py` - Core logic module containing performance algorithms and automated feedback generation.
* `requirements.txt` - Python dependencies needed to host and run the application.
* `.streamlit/config.toml` - Custom UI styling and color palette configuration.
